Quick Reference

These PDFs are lookup guides while coding. They are intentionally separate from the two main notebooks.

ctype.h

Character classification and case conversion.

Open ctype.h PDF
  • Checks: isalnum, isalpha, isdigit, isxdigit, islower, isupper
  • Whitespace and printable checks: isspace, isprint, isgraph, iscntrl, ispunct
  • Case conversion: tolower, toupper

string.h

Core C string copy, concat, and compare tools.

Open string.h PDF
  • Copy: strcpy, strncpy
  • Concatenate: strcat, strncat
  • Compare: strcmp, strncmp

Header references are separate by design for quick usage while coding.
